**SchmancyBox** a Lightbox solution (FancyBox), built with jQuery. Customizable from the admin area.

[More Information & Feature Requests](http://geenius.co.uk/apps/schmancy-box-a-wordpress-plugin/) |
[How-To Guide](http://www.geenius.co.uk/wp-content/resources/schmancy/schmancy-box-how-to.pdf)|
[Original jQuery Lightbox](http://fancy.klade.lv/)|

Some features:

* Works with images and iFrame content
* Applied easily by adding a class to any link
* Works as a gallery
* Easily customisable through admin area, Define Default iFrame Width/Height, Overlay opacity etc.
* Tested in 2.6 and 2.7

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory. If you're uploading it make sure to upload
the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Define your SchmancyBox settings in the WordPress admin menu
1. That's it!
