== Description ==
This Plugin help you to insert easy and clean Google analytics tracking code and preview simple statistics in your wordpres dashboard

== Installation ==
1. Copy plugin folder (dashboard-google-analytics) into wp-content\plugins\
2. Go To your wordpress dashboard and activate plugin Dashboard Google Analytics
3. Plugin interface can be found on Tools sections

== How to uninstall Dashboard Google Analytics ==
1. Deactivate plugin Dashboard Google Analytics
2. Delete folder dashboard-google-analytics from wp-content\plugins\
